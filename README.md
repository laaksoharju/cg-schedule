# cg-schedule
Simple interval scheduling app

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Browser

When server is running, visit the following page in a web browser:

http://localhost:8080/
